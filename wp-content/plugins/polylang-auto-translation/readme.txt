Add auto-translation functionnality to the Polylang plugin.

== Description ==

Add a meta box to make automatic translation of the article and page content using free webservices.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `polylang-auto-translation` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
